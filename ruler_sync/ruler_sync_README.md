# RulerSync 雙螢幕比例尺同步工具

PACS 閱片時用來比對兩顆螢幕的縮放比例尺是否一致。

## 需求

- Python 3.x（Windows 內建 ctypes，無需安裝額外套件）

## 使用方式

```bash
python ruler_sync.py
```

## 操作說明

| 操作 | 效果 |
|------|------|
| 上半部左鍵拖曳 | 移動尺的位置（Y 軸同步，X 軸獨立） |
| 底部色條左鍵拖曳 | 縮放高度（兩條尺同步） |
| 右鍵 | 關閉程式 |

## 設定

開啟 `ruler_sync.py`，修改頂部設定區：

```python
RULER_W     = 44      # 尺的寬度（px）
RULER_MIN_H = 80      # 最小高度（px）
RESIZE_ZONE = 14      # 底部拖曳縮放區高度（px）
TICK_BIG    = 100     # 大刻度間距（px）
TICK_MID    = 50      # 中刻度間距（px）
TICK_SM     = 10      # 小刻度間距（px）
INIT_H      = 400     # 初始高度（px）

A_X, A_Y    = 20,   100    # A 尺初始位置（第一螢幕）
B_X, B_Y    = 1940, 100    # B 尺初始位置（第二螢幕，X = 第一螢幕寬度）
```

> `B_X` 通常等於第一螢幕的水平解析度（例如 1920px 螢幕則設為 `1920`）。
