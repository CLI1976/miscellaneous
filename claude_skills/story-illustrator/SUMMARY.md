# Story Illustrator Skill - 快速總結

## 🎯 這個 Skill 做什麼？

**自動為故事文章生成連貫、角色一致的插圖**

### 核心功能

1. **智能場景分析** 📊
   - 自動判斷需要幾張圖
   - 識別關鍵情節點
   - 評分選擇最佳場景

2. **角色一致性控制** 👤
   - 生成角色設定圖 (Character Sheet)
   - 設計特色附屬品 (Unique Accessory) ⭐ 新增
   - 確保每張圖中角色外觀一致

3. **多語言標籤支援** 🌐 ⭐ 新增
   - `--lang TW` 繁體中文標籤
   - `--lang EN` 英文標籤
   - 控制圖片中的文字語言

4. **專業提示詞生成** ✍️
   - 詳細的圖片描述
   - 適合 AI 圖像生成
   - 可直接用於 DALL-E / Midjourney / Stable Diffusion

## ⚡ 快速開始

### 1. 安裝

```bash
# 複製到 Claude Code 目錄
cp -r story-illustrator ~/.claude/skills/
```

### 2. 使用

```bash
# 分析故事，規劃場景
/story-illustrator your-story.md --plan-only

# 確認後生成插圖
yes
```

### 3. 輸出

```
output/
├── character-sheet-main.png  # ⭐ 角色設定圖
├── characters.json           # 角色設定
├── scenes-plan.md            # 場景規劃
├── scene-01.png              # 插圖 1
├── scene-02.png              # 插圖 2
└── ...
```

## 💡 關鍵特色

### ✅ 解決的問題

**問題 1**: AI 生成的角色每張圖都不同  
**解決**: 建立詳細的角色描述模板，每次生成都用相同描述

**問題 2**: 不知道該為哪些段落配圖  
**解決**: 自動評分系統，選擇視覺性高、情感強的場景

**問題 3**: 手動寫提示詞太費時  
**解決**: 自動組裝角色+場景+風格的完整提示詞

### ⚙️ 設計理念

1. **分階段** - 先規劃後生成，避免浪費
2. **可調整** - 每個環節都可人工介入
3. **模組化** - 可獨立使用場景分析或角色管理
4. **通用性** - 不依賴特定圖片生成工具

## 🔧 核心技術

### 場景識別邏輯

根據以下標準判斷場景：
- 時間變化
- 地點切換
- 情節轉折
- 情緒變化
- 段落結構

### 角色一致性方法

```json
{
  "settings": {
    "lang": "TW",
    "style": "storybook"
  },
  "角色名": {
    "description_template": "完整英文描述",
    "hair": "髮型",
    "eyes": "眼睛",
    "clothing": "衣著",
    "unique_accessory": "特色附屬品（眼鏡、髮帶等）",
    ...
  }
}
```

**角色一致性保障**：
1. 生成角色設定圖作為視覺參考
2. 為主角設計特色附屬品增加辨識度
3. 每次生成都使用完全相同的 `description_template`

### 評分系統

- 視覺性 (30%) - 是否有清晰視覺元素
- 重要性 (40%) - 是否是關鍵情節
- 情感強度 (20%) - 是否有強烈情感
- 代表性 (10%) - 是否能代表章節

## ⚠️ 重要提示

### 圖片生成工具

這個 skill **不包含**圖片生成功能，你需要：

**選項 1**: 在 claude.ai 使用（推薦）
- Claude.ai 有內建圖片生成
- 可以直接調用

**選項 2**: 使用外部工具
- 安裝 baoyu-gemini-web
- 或其他 MCP 圖片生成插件

**選項 3**: 手動生成
- Skill 輸出提示詞
- 你用其他工具生成

### 角色一致性限制

AI 生成圖片天生有變異性：
- 即使用相同提示詞，仍可能有差異
- 建議生成後人工檢查
- 必要時重新生成個別場景
- 第一張圖最關鍵（會作為參考）

## 🚀 進階功能

### 自訂風格

編輯 `EXTEND.md` 添加新風格：

```markdown
## 新風格: cyberpunk
- 賽博龐克風格
- 霓虹燈、未來感
- 提示詞: "cyberpunk style, neon lights, futuristic"
```

### 調整評分權重

修改 `scene-analyzer.md`:

```markdown
視覺性: 30% → 50%  (更重視視覺表現)
重要性: 40% → 30%
```

### 手動選擇場景

```bash
# 僅生成指定場景
/story-illustrator story.md --scenes 1,3,5,8
```

## 🤝 與 baoyu-comic 的差異

| 特性 | story-illustrator | baoyu-comic |
|------|-------------------|-------------|
| 場景分析 | ✅ 自動化評分系統 | ⚠️ 基本分析 |
| 角色一致性 | ✅ 詳細檔案系統 | ⚠️ 簡單描述 |
| 分階段確認 | ✅ 先規劃後生成 | ❌ 直接生成 |
| 可調整性 | ✅ 高度可自訂 | ⚠️ 有限 |
| 圖片生成 | ⚠️ 需外部工具 | ✅ 內建 |

**總結**: 
- `story-illustrator` = 更專業、更可控、更適合重要項目
- `baoyu-comic` = 更快速、更簡單、適合快速原型
