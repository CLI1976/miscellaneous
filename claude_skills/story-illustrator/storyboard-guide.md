# Storyboard Guide - 分鏡腳本指南

為每個場景生成專業的分鏡腳本，包含鏡頭、構圖、情緒追蹤和轉場建議。

## 功能概覽

### 1. 分鏡腳本 (Storyboard)
- 鏡頭類型與角度
- 構圖與角色位置
- 動作描述與時間軸

### 2. 情緒追蹤 (Emotion Tracking)
- 角色情緒變化曲線
- 場景情緒基調
- 情緒轉折點標記

### 3. 轉場建議 (Scene Transition)
- 場景間的視覺連接
- 轉場類型建議
- 敘事節奏控制

---

## 一、分鏡腳本系統

### 鏡頭類型 (Shot Types)

| 類型 | 英文 | 用途 | 適用場景 |
|------|------|------|----------|
| 遠景 | Wide Shot (WS) | 展示環境、建立場景 | 開場、地點轉換 |
| 全景 | Full Shot (FS) | 展示角色全身 | 角色登場、動作場景 |
| 中景 | Medium Shot (MS) | 角色腰部以上 | 對話、互動 |
| 近景 | Close-Up (CU) | 臉部或物品特寫 | 情緒表達、重要細節 |
| 特寫 | Extreme Close-Up (ECU) | 眼睛、手部等 | 強烈情緒、關鍵物品 |
| 雙人鏡頭 | Two Shot | 兩個角色同框 | 對話、關係展現 |
| 過肩鏡頭 | Over-the-Shoulder (OTS) | 從一方肩膀看另一方 | 對話、觀點表達 |

### 鏡頭角度 (Camera Angles)

| 角度 | 英文 | 效果 | 適用情境 |
|------|------|------|----------|
| 平視 | Eye Level | 中性、自然 | 一般敘事 |
| 俯視 | High Angle | 角色顯得弱小、脆弱 | 挫敗、無助 |
| 仰視 | Low Angle | 角色顯得強大、威嚴 | 勝利、決心 |
| 鳥瞰 | Bird's Eye | 全局視角 | 環境展示、戰略場景 |
| 傾斜 | Dutch Angle | 不安、緊張 | 衝突、危機 |

### 構圖法則 (Composition)

| 構圖 | 說明 | 適用場景 |
|------|------|----------|
| 三分法 | 主體置於三分線交叉點 | 通用，平衡感 |
| 中心構圖 | 主體置中 | 強調、對稱場景 |
| 框中框 | 用門窗等框住主體 | 孤立、聚焦 |
| 引導線 | 線條引導視線至主體 | 動態、方向感 |
| 對角線 | 主體沿對角線排列 | 動感、張力 |
| 負空間 | 大量留白 | 孤獨、思考 |

### 分鏡腳本模板

```markdown
## 場景 X: [場景標題]

### 基本資訊
- **時間**: [故事中的時間]
- **地點**: [場景地點]
- **角色**: [出場角色]
- **情緒基調**: [主要情緒]

### 分鏡設定
- **鏡頭類型**: Medium Shot (MS) / 中景
- **鏡頭角度**: Eye Level / 平視
- **構圖**: 三分法，主角置於右側三分線

### 角色位置
```
┌─────────────────────────┐
│                         │
│     [角色B]             │
│         ↘              │
│           [角色A] ←主角 │
│                         │
│   ═══桌子═══            │
│                         │
└─────────────────────────┘
```

### 動作描述
1. 角色A 坐在桌前，表情緊張
2. 角色B 站在一旁，手放在A的肩膀上
3. 窗外陽光灑入，形成溫暖氛圍

### 視覺重點
- 角色A 的表情變化（從緊張到釋然）
- 兩人的肢體互動
- 桌上的 [重要道具]

### 情緒標記
- 角色A: 緊張 → 感動 (情緒轉折)
- 角色B: 關心、鼓勵

### 轉場至下一場景
- **轉場類型**: 時間跳躍
- **視覺連接**: 同一地點，光線從午後變為傍晚
```

## 二、角色情緒追蹤系統

### 情緒分類

#### 基本情緒 (6 種)
| 情緒 | 英文 | 視覺表現 | 強度等級 |
|------|------|----------|----------|
| 快樂 | Joy | 笑容、明亮眼神 | 1-5 |
| 悲傷 | Sadness | 淚水、低垂眼簾 | 1-5 |
| 憤怒 | Anger | 皺眉、緊繃表情 | 1-5 |
| 恐懼 | Fear | 睜大眼睛、退縮 | 1-5 |
| 驚訝 | Surprise | 張大嘴巴、揚眉 | 1-5 |
| 厭惡 | Disgust | 皺鼻、撇嘴 | 1-5 |

#### 複合情緒
| 情緒 | 組成 | 視覺表現 |
|------|------|----------|
| 感動 | 快樂 + 悲傷 | 含淚微笑 |
| 焦慮 | 恐懼 + 悲傷 | 咬唇、不安眼神 |
| 羞愧 | 悲傷 + 恐懼 | 低頭、臉紅 |
| 驕傲 | 快樂 + 憤怒 | 昂首、自信笑容 |
| 嫉妒 | 憤怒 + 悲傷 | 斜視、緊抿嘴唇 |
| 期待 | 快樂 + 驚訝 | 閃亮眼神、前傾 |

### 情緒追蹤表格

```markdown
## 角色情緒追蹤表

### 角色: 政彬

| 場景 | 情緒 | 強度 | 表情描述 | 觸發事件 |
|------|------|------|----------|----------|
| 1 | 緊張+壓力 | 4 | 皺眉、握緊拳頭 | 比賽即將開始 |
| 2 | 不滿+委屈 | 5 | 抱胸、撇嘴 | 被換掉棒次 |
| 3 | - | - | (未出場) | - |
| 4 | 慚愧+感動 | 4 | 低頭、眼眶泛紅 | 子強的鼓勵 |
| 5 | 堅定+熱血 | 5 | 眼神銳利、嘴角上揚 | 決心全力以赴 |
| 6 | 興奮+釋然 | 5 | 大笑、張開雙臂 | 勝利擁抱 |

### 情緒曲線

```
強度
5 │    ╭─╮                    ╭─────╮
4 │  ╭─╯ │              ╭─────╯     │
3 │  │   │              │           │
2 │  │   │              │           │
1 │  │   ╰──────────────╯           │
0 │──┴─────────────────────────────────
    場景 1   2   3   4   5   6
```

### 情緒轉折點
- **場景 2→4**: 從「不滿」到「慚愧」（關鍵轉折）
- **場景 4→5**: 從「慚愧」到「堅定」（角色成長）
- **場景 5→6**: 從「堅定」到「興奮」（高潮釋放）
```

### 情緒在提示詞中的應用

```
Characters present:
- Zhengbin: East Asian boy age 12, short messy black hair,
  WEARING RED WRISTBAND (identifying feature),
  EMOTION: determined and focused, eyes sharp with resolve,
  slight confident smile, athletic stance ready to sprint
```

## 三、場景轉場系統

### 轉場類型

#### 1. 時間轉場
| 類型 | 說明 | 視覺提示 |
|------|------|----------|
| 直接跳躍 | 時間直接切換 | 光線變化、鐘錶 |
| 漸進過渡 | 展示時間流逝 | 日落、月亮升起 |
| 回憶/閃回 | 進入過去 | 模糊邊緣、褪色 |

#### 2. 空間轉場
| 類型 | 說明 | 視覺提示 |
|------|------|----------|
| 地點切換 | 場景位置改變 | 新環境建立鏡頭 |
| 移動轉場 | 角色移動中 | 走廊、道路 |
| 同場景換位 | 同地點不同視角 | 鏡頭旋轉感 |

#### 3. 情緒轉場
| 類型 | 說明 | 視覺提示 |
|------|------|----------|
| 對比轉場 | 情緒反差 | 色調對比 |
| 漸變轉場 | 情緒逐漸變化 | 色彩漸變 |
| 衝擊轉場 | 突發事件 | 動態模糊 |

#### 4. 敘事轉場
| 類型 | 說明 | 視覺提示 |
|------|------|----------|
| 因果連接 | 前因後果 | 動作延續 |
| 平行敘事 | 同時發生 | 分割畫面感 |
| 主題連接 | 相似元素 | 物品/顏色呼應 |

### 轉場建議模板

```markdown
## 場景轉場分析

### 場景 1 → 場景 2
- **轉場類型**: 時間跳躍 + 回憶
- **時間跨度**: 幾天前
- **空間變化**: 操場 → 體育館
- **情緒變化**: 緊張 → 不滿
- **視覺連接建議**:
  - 場景1 結尾：政彬握著接力棒的手
  - 場景2 開頭：同樣的手，但場景是訓練時
- **色調建議**:
  - 場景1：明亮、高對比（比賽氛圍）
  - 場景2：稍暗、回憶感（過去事件）

### 場景 2 → 場景 3
- **轉場類型**: 時間跳躍
- **時間跨度**: 比賽當天
- **空間變化**: 體育館 → 跳高場地
- **情緒變化**: 政彬的不滿 → 名揚的受傷
- **視覺連接建議**:
  - 場景2 結尾：政彬不滿的表情
  - 場景3 開頭：名揚痛苦的表情（情緒對比）
- **敘事作用**: 製造轉折，讓政彬開始反思
```

### 轉場連貫性檢查

```markdown
## 轉場連貫性檢查表

| 轉場 | 時間邏輯 | 空間邏輯 | 情緒連貫 | 視覺連接 | 評分 |
|------|----------|----------|----------|----------|------|
| 1→2 | ✓ 合理 | ✓ 合理 | ✓ 有對比 | ⚠️ 需加強 | 4/5 |
| 2→3 | ✓ 合理 | ✓ 合理 | ✓ 轉折點 | ✓ 表情對比 | 5/5 |
| 3→4 | ✓ 合理 | ✓ 同場地 | ✓ 連續 | ✓ 自然 | 5/5 |
| 4→5 | ✓ 合理 | ✓ 同場地 | ✓ 成長 | ✓ 姿態對比 | 5/5 |
| 5→6 | ✓ 合理 | ✓ 終點線 | ✓ 高潮 | ✓ 動態連接 | 5/5 |
```

## 四、完整分鏡腳本範例

### 故事：接力賽

```markdown
# 分鏡腳本：接力賽

## 故事資訊
- **標題**: 接力賽
- **風格**: storybook
- **場景數**: 6
- **主角**: 政彬、名揚

---

## 場景 1: 起跑線前的壓力

### 基本資訊
- **時間**: 比賽當天下午
- **地點**: 學校操場，起跑線
- **角色**: 政彬
- **情緒基調**: 緊張、壓力

### 分鏡設定
- **鏡頭類型**: Medium Close-Up (MCU)
- **鏡頭角度**: Slightly Low Angle（輕微仰視，強調壓力感）
- **構圖**: 三分法，政彬置於畫面左側，右側留空展示跑道延伸

### 角色位置圖
```
┌─────────────────────────────┐
│  ☀️                         │
│                    ～～～～  │
│ [政彬]              跑道    │
│  緊握                延伸→  │
│  接力棒                     │
│ ════起跑線════              │
└─────────────────────────────┘
```

### 動作描述
1. 政彬站在起跑位置，身體微微前傾
2. 雙手緊握接力棒，指節發白
3. 眼神凝視前方跑道，眉頭微皺
4. 背景：其他選手的模糊身影

### 情緒標記
- **政彬**: 緊張 (4/5)、壓力 (4/5)
- **表情**: 皺眉、抿唇、眼神專注但帶有不安

### 視覺重點
- 政彬緊握接力棒的手（特寫元素）
- 向前延伸的跑道（象徵未知挑戰）
- 明亮的陽光（比賽氛圍）

### 色調建議
- 主色調：明亮暖色（陽光）
- 強調色：跑道的紅色
- 情緒色：略帶緊張的高對比

### 轉場至場景 2
- **轉場類型**: 時間回溯（回憶）
- **視覺連接**: 政彬的手 → 同樣的手在訓練時
- **色調變化**: 明亮 → 稍微褪色（回憶感）

---

## 場景 2: 訓練時的委屈

### 基本資訊
- **時間**: 幾天前，訓練時
- **地點**: 學校體育館/操場邊
- **角色**: 政彬、名揚、老師
- **情緒基調**: 不滿、委屈

### 分鏡設定
- **鏡頭類型**: Medium Shot (MS)
- **鏡頭角度**: Eye Level
- **構圖**: 三角構圖，老師在中間，政彬和名揚在兩側

### 角色位置圖
```
┌─────────────────────────────┐
│      [老師]                 │
│      解釋中                 │
│     ↙    ↘                 │
│ [政彬]    [名揚]           │
│ 抱胸      平靜              │
│ 不滿      (戴眼鏡)          │
└─────────────────────────────┘
```

### 動作描述
1. 老師站在中間，手勢說明調換棒次
2. 政彬雙臂抱胸，表情不滿，身體微微側開
3. 名揚站姿端正，表情平靜，戴著眼鏡

### 情緒標記
- **政彬**: 不滿 (5/5)、委屈 (4/5)
- **名揚**: 平靜 (3/5)、略感尷尬 (2/5)
- **老師**: 耐心 (3/5)

### 視覺重點
- 政彬的防禦姿態（抱胸）
- 三人的位置關係（政彬與其他人的距離感）
- 名揚的眼鏡（特色附屬品）

### 轉場至場景 3
- **轉場類型**: 時間跳躍
- **視覺連接**: 政彬不滿的表情 → 名揚痛苦的表情
- **敘事作用**: 情緒對比，製造轉折

---

## 場景 6: 衝線勝利擁抱

### 基本資訊
- **時間**: 比賽結束瞬間
- **地點**: 終點線
- **角色**: 政彬、名揚
- **情緒基調**: 興奮、感動、和解

### 分鏡設定
- **鏡頭類型**: Medium Shot (MS)
- **鏡頭角度**: Slightly Low Angle（仰視，強調勝利感）
- **構圖**: 中心構圖，兩人擁抱置於畫面中央

### 角色位置圖
```
┌─────────────────────────────┐
│    🎊  VICTORY!  🎊        │
│         RELAY RACE          │
│                             │
│      [政彬 ⟷ 名揚]         │
│        擁抱中               │
│        (名揚戴眼鏡)         │
│                             │
│  👥👥  歡呼的同學  👥👥    │
└─────────────────────────────┘
```

### 動作描述
1. 政彬衝向名揚，張開雙臂
2. 兩人緊緊擁抱，臉上都是燦爛笑容
3. 名揚的眼鏡略微歪斜（生動細節）
4. 背景：歡呼的同學、飄揚的彩帶

### 情緒標記
- **政彬**: 興奮 (5/5)、釋然 (5/5)、感動 (4/5)
- **名揚**: 喜悅 (5/5)、欣慰 (4/5)

### 視覺重點
- 兩人的擁抱姿態（和解的象徵）
- 真誠的笑容
- 名揚歪斜的眼鏡（生動細節）
- 背景的歡慶氛圍

### 色調建議
- 主色調：溫暖金色（勝利的陽光）
- 強調色：藍白運動服
- 情緒色：明亮、高飽和度（歡樂氛圍）

### 結束
- 這是最後一個場景
- 情緒達到最高點
- 故事完美收尾
```

## 六、在提示詞中的應用

### 整合分鏡資訊的提示詞

```
Storybook illustration, Medium Close-Up shot, slightly low angle.

Scene: A nervous boy at relay race starting line.

COMPOSITION: Rule of thirds, boy on left side, track extending to the right.

CHARACTER (MUST match reference):
- Zhengbin: East Asian boy age 12, short messy black hair,
  WEARING RED WRISTBAND (identifying feature),
  EMOTION: nervous and pressured (intensity 4/5),
  expression shows furrowed brows, tightly pressed lips,
  hands gripping relay baton with white knuckles

SETTING: School track field, starting line, bright afternoon sunlight,
other runners blurred in background

MOOD: Tense, pressured, anticipation
LIGHTING: Bright sunlight from above, high contrast
COLOR TONE: Warm but with tension, red track prominent

Style: Children's storybook illustration, soft warm colors.
```
