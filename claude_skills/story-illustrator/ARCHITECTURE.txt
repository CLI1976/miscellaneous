# Story Illustrator - 系統架構圖

```
┌────────────────────────────────────────────────────────────────────────┐
│                          Story Illustrator Skill                        │
│                                                                          │
│  輸入: story.md (故事文章)                                                │
└────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌────────────────────────────────────────────────────────────────────────┐
│                      階段 1: 分析與規劃                                    │
│                                                                          │
│  ┌─────────────────────┐      ┌──────────────────────┐                │
│  │  Scene Analyzer     │      │  Character Extractor │                │
│  │  (scene-analyzer.md)│      │  (character-guide.md)│                │
│  │                     │      │                      │                │
│  │  • 識別場景切換信號   │      │  • 提取角色特徵      │                │
│  │  • 評分排序         │      │  • 建立一致性模板    │                │
│  │  • 選擇最佳場景     │      │  • 生成角色檔案      │                │
│  └─────────────────────┘      └──────────────────────┘                │
│           │                              │                              │
│           └──────────────┬───────────────┘                              │
│                          ▼                                              │
│              ┌───────────────────────┐                                  │
│              │   Planning Report     │                                  │
│              │  (scenes-plan.md)     │                                  │
│              │                       │                                  │
│              │  • 場景清單           │                                  │
│              │  • 角色設定           │                                  │
│              │  • 提示詞草稿         │                                  │
│              └───────────────────────┘                                  │
└────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
                        ┌───────────────────────┐
                        │   User Confirmation   │
                        │   (yes/no/modify)     │
                        └───────────────────────┘
                                    │
                                    ▼ yes
┌────────────────────────────────────────────────────────────────────────┐
│                      階段 2: 生成插圖                                      │
│                                                                          │
│  ┌──────────────────────────────────────────────────────┐              │
│  │            Prompt Assembly Engine                    │              │
│  │                                                      │              │
│  │  For each scene:                                    │              │
│  │  1. Load character templates from characters.json  │              │
│  │  2. Combine: Scene + Characters + Style            │              │
│  │  3. Generate complete prompt (300+ words)          │              │
│  └──────────────────────────────────────────────────────┘              │
│                              │                                          │
│                              ▼                                          │
│  ┌──────────────────────────────────────────────────────┐              │
│  │           Image Generation Layer                     │              │
│  │                                                      │              │
│  │  Option 1: Claude.ai built-in                       │              │
│  │  Option 2: External API (Gemini/DALL-E)             │              │
│  │  Option 3: Manual (export prompts only)             │              │
│  └──────────────────────────────────────────────────────┘              │
│                              │                                          │
│                              ▼                                          │
│  ┌──────────────────────────────────────────────────────┐              │
│  │            Quality Check & Retry                     │              │
│  │                                                      │              │
│  │  • Verify generation success                        │              │
│  │  • Handle errors                                    │              │
│  │  • Retry if needed                                  │              │
│  └──────────────────────────────────────────────────────┘              │
└────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌────────────────────────────────────────────────────────────────────────┐
│                           輸出檔案                                        │
│                                                                          │
│  output/                                                                │
│  ├── characters.json          ← 角色設定檔                              │
│  ├── scenes-plan.md          ← 場景規劃文件                             │
│  ├── scene-01.png            ← 生成的插圖                               │
│  ├── scene-02.png                                                       │
│  ├── ...                                                                │
│  └── generation-summary.md   ← 生成摘要報告                             │
└────────────────────────────────────────────────────────────────────────┘


═══════════════════════════════════════════════════════════════════════════
                            核心組件詳解
═══════════════════════════════════════════════════════════════════════════

┌────────────────────────────────────────────────────────────────────────┐
│  Scene Analyzer (場景分析器)                                              │
├────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  輸入: 完整文章文本                                                        │
│                                                                          │
│  步驟 1: 初步劃分                                                         │
│  ┌──────────────────────────────────────┐                              │
│  │ • 掃描時間變化標記                    │                              │
│  │ • 掃描地點切換標記                    │                              │
│  │ • 掃描情節轉折標記                    │                              │
│  │ → 生成 N 個候選場景 (N = 1.5x 目標)  │                              │
│  └──────────────────────────────────────┘                              │
│                                                                          │
│  步驟 2: 評分系統                                                         │
│  ┌──────────────────────────────────────┐                              │
│  │ 視覺性 (30%)                          │                              │
│  │ • 是否有具體視覺元素?                 │                              │
│  │ • 場景是否容易繪製?                   │                              │
│  │                                      │                              │
│  │ 重要性 (40%)                          │                              │
│  │ • 是否是關鍵情節點?                   │                              │
│  │ • 是否推動故事發展?                   │                              │
│  │                                      │                              │
│  │ 情感強度 (20%)                        │                              │
│  │ • 是否有強烈情感?                     │                              │
│  │ • 讀者是否會產生共鳴?                 │                              │
│  │                                      │                              │
│  │ 代表性 (10%)                          │                              │
│  │ • 是否能代表章節?                     │                              │
│  │ • 是否包含主要角色?                   │                              │
│  └──────────────────────────────────────┘                              │
│                                                                          │
│  步驟 3: 選擇與驗證                                                       │
│  ┌──────────────────────────────────────┐                              │
│  │ • 按總分排序                          │                              │
│  │ • 檢查分佈均勻性                      │                              │
│  │ • 確保涵蓋開頭/中段/結尾              │                              │
│  │ • 驗證故事弧線完整性                  │                              │
│  └──────────────────────────────────────┘                              │
│                                                                          │
│  輸出: 排序的場景列表 + 評分                                               │
└────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────┐
│  Character Extractor (角色提取器)                                         │
├────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  輸入: 完整文章文本                                                        │
│                                                                          │
│  步驟 1: 識別角色                                                         │
│  ┌──────────────────────────────────────┐                              │
│  │ • 尋找人名                            │                              │
│  │ • 識別主要角色 vs 配角                │                              │
│  │ • 統計出現頻率                        │                              │
│  └──────────────────────────────────────┘                              │
│                                                                          │
│  步驟 2: 提取特徵                                                         │
│  ┌──────────────────────────────────────┐                              │
│  │ 外貌:                                 │                              │
│  │ • 頭髮 (顏色、長度、樣式)             │                              │
│  │ • 眼睛 (顏色)                         │                              │
│  │ • 臉型、身材                          │                              │
│  │                                      │                              │
│  │ 衣著:                                 │                              │
│  │ • 風格 (維多利亞、現代、傳統)         │                              │
│  │ • 顏色                                │                              │
│  │ • 標誌性配件                          │                              │
│  │                                      │                              │
│  │ 個性:                                 │                              │
│  │ • 常見表情                            │                              │
│  │ • 姿態                                │                              │
│  │ • 性格特質                            │                              │
│  └──────────────────────────────────────┘                              │
│                                                                          │
│  步驟 3: 生成模板                                                         │
│  ┌──────────────────────────────────────┐                              │
│  │ {                                    │                              │
│  │   "角色名": {                         │                              │
│  │     "description_template": "...",   │                              │
│  │     "hair": "...",                   │                              │
│  │     "eyes": "...",                   │                              │
│  │     ...                              │                              │
│  │   }                                  │                              │
│  │ }                                    │                              │
│  └──────────────────────────────────────┘                              │
│                                                                          │
│  輸出: characters.json                                                   │
└────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────┐
│  Prompt Assembly Engine (提示詞組裝引擎)                                  │
├────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  輸入: Scene Data + Characters JSON + Style Params                       │
│                                                                          │
│  組裝結構:                                                                │
│  ┌──────────────────────────────────────────────────┐                  │
│  │ [開頭] 風格聲明                                    │                  │
│  │ "A [style] illustration showing [action]"        │                  │
│  │                                                  │                  │
│  │ [角色部分] 完整角色描述                            │                  │
│  │ "Characters visible in this scene:"              │                  │
│  │ "- [Name]: [FULL description_template]"          │                  │
│  │ "- [Name]: [FULL description_template]"          │                  │
│  │                                                  │                  │
│  │ [場景部分] 環境描述                                │                  │
│  │ "Setting: [environment details]"                 │                  │
│  │                                                  │                  │
│  │ [情緒部分] 氛圍描述                                │                  │
│  │ "Mood: [emotional tone]"                         │                  │
│  │                                                  │                  │
│  │ [光線部分] 光線效果                                │                  │
│  │ "Lighting: [light description]"                  │                  │
│  │                                                  │                  │
│  │ [構圖部分] 視覺構圖                                │                  │
│  │ "Composition: [layout description]"              │                  │
│  │                                                  │                  │
│  │ [結尾] 風格確認                                    │                  │
│  │ "Art style: [style] suitable for [audience]"     │                  │
│  └──────────────────────────────────────────────────┘                  │
│                                                                          │
│  關鍵原則:                                                                │
│  • 每次都用完整的 description_template                                   │
│  • 不省略任何角色細節                                                     │
│  • 保持描述順序一致                                                       │
│  • 只描述本場景出現的角色                                                 │
│                                                                          │
│  輸出: 300+ 字的完整提示詞                                                │
└────────────────────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════════════════
                            擴展點
═══════════════════════════════════════════════════════════════════════════

1. 自訂場景分析規則
   └─→ 編輯 scene-analyzer.md
       └─→ 修改評分權重
       └─→ 添加新的切換信號

2. 自訂角色一致性規則
   └─→ 編輯 character-guide.md
       └─→ 添加新的特徵類型
       └─→ 調整模板格式

3. 添加新風格
   └─→ 創建 EXTEND.md
       └─→ 定義新的風格參數
       └─→ 指定提示詞模板

4. 整合新的圖片生成工具
   └─→ 修改 SKILL.md
       └─→ 添加新的 API 調用邏輯
       └─→ 處理新的輸出格式

5. 添加後處理步驟
   └─→ 圖片品質檢查
   └─→ 自動重試
   └─→ 風格一致性驗證
